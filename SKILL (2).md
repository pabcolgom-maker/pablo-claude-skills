# Materiales para Compradores Internacionales de Café

## Contexto de origen

- **Región principal:** Bruselas, Pitalito, Huila — Colombia
- **Tipo:** Café de especialidad (specialty coffee)
- **Relación:** Pablo trabaja directamente con productores de la zona
- **Mercado objetivo:** Compradores internacionales (importadores, tostadores, distribuidores)

## Tipos de documentos frecuentes

### 1. Buyer Information Form (Formulario de información del comprador)
Campos estándar:
- Company name / Nombre empresa
- Contact person / Persona de contacto
- Country / País
- Annual volume needed (kg/bags) / Volumen anual
- Certifications required / Certificaciones requeridas
- Processing preferences / Preferencias de proceso
- Cupping score minimum / Puntaje mínimo de cata
- Preferred contact method

### 2. Lot Profile / Ficha de Lote
Estructura estándar:
- Productor / Producer
- Variedad / Variety
- Proceso / Process (washed, natural, honey, etc.)
- Altitud / Altitude (msnm / masl)
- Cosecha / Harvest
- Puntaje SCA / SCA Score
- Perfil sensorial / Sensory profile (notas de cata)
- Disponibilidad / Availability (sacos/kg)
- Precio / Price (FOB Buenaventura u otro Incoterm)

### 3. PDF Premium de Presentación de Origen
- Diseño premium, A4 landscape o portrait
- Fotos de finca, proceso, comunidad
- Mapa de la región
- Historia del productor
- Perfiles de lotes disponibles

### 4. Cupping Report
- Fecha de cata
- Catador / Evaluator
- Tabla SCA con puntuaciones por atributo
- Notas descriptivas

## Estándares de calidad en los documentos

- **Idioma:** Español + Inglés (bilingüe) en la mayoría de documentos para compradores
- **Unidades:** kg y sacos de 70kg (indicar ambas siempre que sea posible)
- **Incoterm por defecto:** FOB Buenaventura, salvo que Pablo indique otro
- **Certificaciones comunes a mencionar si aplica:** RFA, UTZ, Orgánico, Bird Friendly
- **Puntaje mínimo de especialidad:** 80 puntos SCA (mencionar si el lote supera 85 o 87)

## Tono y estilo

- Profesional, premium, orientado a exportación
- Para documentos bilingües: español primero, inglés segundo, o en columnas paralelas
- Resaltar la identidad del Huila como región de origen de clase mundial
- Lenguaje sensorial preciso (acidez, cuerpo, dulzura, notas florales/frutales/achocolatadas)

## Instrucciones al usar este skill

1. Si el documento es para un comprador específico, preguntar nombre del comprador y país
2. Para lotes de café, siempre pedir: variedad, proceso, altitud, cosecha, puntaje SCA
3. Si se necesita diseño premium (PDF), leer también el skill `pdf`
4. Preguntar si se incluyen fotos — si sí, Pablo las subirá directamente
5. Para formularios interactivos, crear en formato PDF fillable o HTML
6. Mantener datos de contacto de Pablo como punto de intermediación comercial
